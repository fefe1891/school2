What is the difference between var and let:

Var can be reassigned and redeclared. Let you can only reassign. A hoisted variable can be accessed with var and let creates a block scope.

What is the difference between var and const:

You can reassign & redeclare with var. Const cannot be reassigned or redeclared.You can also hoist a variable with var while const is a block scope

What is the difference between let & const:

You can reassign with both but only redeclare with let

What is hoisting:

This is how the js compiler is explained. Variables are hoisted(lifted) to the top of the scope they are declared in. When using var you can access the variable name & and it's undefined value before being later used. function declarations are also hoisted and can be invoked before they are defined in a code base
